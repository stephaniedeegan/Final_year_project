Turing Machines

A Turing Machine is a mathematical model of computation that defines an abstract machine. It consists of a read/write head with a paper tape passing through it
The Turing Machine was invented by Alan Turing.

The tape is of unbounded length and divided into squares.
It serves both as the means of input and output and also as a working memory for storing the results of intermediate steps of the computation.
There must be a finite number of symbols used in the alphabet that the Turing machine can recognise.

There are 6 fundamental operations that a Turing machine performs :

  • Read the symbol that the head is currently over
  • Write a symbol on the square the head is currently over
  • Move the tape left one position
  • Move the tape right one position
  • Change state
  • Halt
An instruction table for a Turing machine is a finite collection of instructions.

Each instruction is of the form :

If the current state is n and the symbol under the head is x, then write y on the square under the head, go to state m, and move one square left or right.

There are three special states :

  • Start state
  • Accept state
  • Reject state
The Turing Machine computes until it produces an output.
If it never enters an accepting or rejecting state the Turing Machine goes on forever.
The purpose of designated halt states is to indicate when the machine has finished computing.

Definition of TM is T = (Q,Σ,┌,q0,δ).
  • Q is the finite set of states
  • Σ represents the input alphabet
  • ┌ is the tape alphabet
  • q0 is the starting state
  • δ is the transition function
The Church-Turing Thesis states that the capabilities of Turing machines are equivalent to those of any other computational device.
