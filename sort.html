Sorting and Searching

Bubble Sort : An algorithm for arranging the elements in an array so that they are in numerical order (ascending or descending).
BubbleSort is the simplies sorting algorithm.

  • Compare the current element to the element on its right
  • If the current element on the left is the larger then we need to swap the values
  • Move on to the next element and repeat the swapping process
  • This process is repeated until the end of the array is reached – at this point the largest item is in the last position of the array
  • We must now return to the first position and repeat the above process starting at Step 1, until all values are in the correct position

public static void BubbleSort(int [] values)
 {
  int limit = values.length;
  int temp = 0;
   for (int i = 0; i < values.length; i++)
  {
   limit --;
   for (int j = 0; j< limit; j++)
   {
     if(values[j] > values[j+1])
     {
      temp = values[j+1];
      values[j+1] = values[j];
      values[j] = temp;
     }
   } // end inner for loop
  } //end outer for loop
 }

Insertion Sort : Insertion sort is a simple sorting algorithm that works by iteratively building a sorted sublist by inserting each unsorted element into its correct position in the sorted sublist.

Quick Sort : works by dividing an array into smaller sub-arrays and then recursively sorting these sub-arrays.

Linear search : a simple searching algorithm used to find an element in a list or an array. It works by sequentially checking each element of the list until a match is found or the end of the list is reached.

Binary search : is a search algorithm designed to find a value in a sorted array.
It works by repeatedly dividing in half the portion of the list that could contain the item, until you've narrowed down the possible locations to just one.
It’s always 2 to the power of something as we halve it (divide it in two).
If the array is SORTED, and is between 1 and 100, it should be possible to find it in 7 goes or less.
Because 2^7 = 128 and is above our upper boundary of 100.

public static int binarySearch(int[] arr, int target) {
  int left = 0;
  int right = arr.length - 1;
  while (left <= right) {
    int mid = (left + right) / 2;
   if (arr[mid] == target) {
     return mid; }
   else if (arr[mid] < target) {
      left = mid + 1; }
   else {
      right = mid - 1; }
 }
  return -1; // Target not found
}

Time complexity

  • Bubble Sort requires O(n^2) to sort
  • Insertion sort requires O(n^2)
  • Quick sort requires O(n log n)
  • Binary Search requires O(Log N)
  • Linear Search requires O(n)
