Inheritance

Inheritance : it is the process of creating a new class by absorbing data and methods from an existing class

Java allows us to reuse class definitions and extend functionality by allowing one class (the child or subclass) to inherit from another class (the parent or superclass).

Superclass : is the class that is extended by another class

All attributes and methods of the superclass are available in each subclass when inheritance is used.
In order to call the super class constructor we use the keyword super().

The keyword extendss allows one class to inherit from another.
This allows us to extend the design of the parent (super) class by enabling us to inherit all its attributes and functionality.
   → class SubClass extends SuperClass { ... }

A subclass can inherit fields and methods from its superclass, as well as add new fields and methods of its own.
A subclass can call a method in its superclass using the super keyword, followed by the method name and arguments.
   → super.methodName(argumentList)

Example Code that implements inheritance :

class Animal {           // superclass
   private String name;
  public Animal(String name) {
    this.name = name;
  }
  public String getName() {
    return name;
  }
   public void eat() {
    System.out.println(name + " is eating");
  }
}
class Dog extends Animal {     // subclass
  public Dog(String name) {
     super(name);                      // call superclass constructor
}
   public void bark() {
    System.out.println(getName() + " is barking");
  }
}

public class Main {          // main program
 public static void main(String[] args) {
// create a Dog object
     Dog dog = new Dog("Roxy");
     dog.eat();
     dog.bark();
  }
}
